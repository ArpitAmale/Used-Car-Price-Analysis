# 🚗 Used Car Price Analysis

## 📌 Project Overview
End-to-end data analysis project on Indian used car market using CarDekho dataset.
Covers complete data analyst workflow from raw messy data to business insights.

## 🎯 Business Problem
What factors drive used car resale prices in India?
Which brands, fuel types, and conditions hold the best value?

## 🛠️ Tech Stack
- Python 3.x
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## 📊 Key Steps Covered
1. Data Loading & First Look
2. Missing Value Detection & Treatment
3. Duplicate Removal
4. String Column Cleaning (mileage, engine, max_power)
5. Outlier Detection & Removal (IQR Method)
6. Feature Engineering (car_age, brand, price_lakhs)
7. Univariate & Bivariate Analysis
8. Brand-wise & Fuel-wise Analysis
9. Correlation Heatmap
10. Depreciation Curve
11. Price Prediction Model (Random Forest)
12. Business Insights & Recommendations

## 📈 Model Performance
- Algorithm : Random Forest Regressor
- R² Score  : ~85–90%
- Key Drivers: max_power, engine size, car age, km driven

## 💡 Key Insights
- Diesel cars hold 30% higher resale value than petrol
- Automatic transmission cars fetch higher prices
- Cars depreciate fastest between year 1–5
- max_power is the single strongest predictor of price

## 📁 Dataset
Source: [CarDekho — Kaggle](https://www.kaggle.com/datasets/nehalbirla/vehicle-dataset-from-cardekho)
